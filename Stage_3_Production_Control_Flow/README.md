## 🧠 Key Concepts Covered
* **Human-in-the-Loop (HITL):** Pausing graph execution for manual approval before sensitive actions (e.g., sending emails or modifying databases).
* **Dynamic Routing:** Dynamically switching between lightweight and heavy LLMs based on the complexity of the ongoing task to optimize cost.
* **State Override:** Manually intervening and altering the agent's memory during execution to correct trajectory.
